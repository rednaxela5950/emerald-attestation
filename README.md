# Emerald + Symbiotic Relay DA
Hackathon prototype for Emerald DA flow anchored to Symbiotic Relay attestation.
## Structure
contracts/ and test/ for Foundry; packages/emerald-da-worker for worker; apps/frontend for UI.
## Setup
Install Foundry (forge) and pnpm >=8, then run `pnpm install`.
Run contract tests with `forge test`.
